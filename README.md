### Calculation Engines


build instructions
```
cd CalculationEngines/
mkdir build
cd build
cmake ..
make
```

Clean the build
```
cd CalculationEngines/
rm -f build
```

Instructions to run the engines with integer input
```
cd build
./src/calc <engine_name> <integer1> <integer2> .....
```

Instructions to run the engines with integer input
```
cd build
./src/calc <engine_name> source <file1> <file2>
```

Instructions to run the unit tests 
```
cd build
./test/cpp-test
```
